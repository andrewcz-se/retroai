AI Arcade Classics

Welcome to a collection of retro arcade games reimagined for the modern web. Each game in this collection was developed as a creative collaboration with Gemini, a large language model from Google. The goal was to explore the capabilities of AI in software development by recreating the look, feel, and fun of timeless arcade classics using modern, single-file web applications.
