Description

Retrofrog is a modern tribute to the classic arcade game Frogger, built entirely in a single HTML file with the Canvas API. Guide Retrofrog across a busy road and a treacherous river to reach the safety of the home slots at the top. With authentic retro visuals, sound effects, and challenging gameplay, it's a nostalgic trip back to the golden age of arcade gaming.

How to Play

The goal is to guide 5 frogs safely into their homes at the top of the screen before the timer runs out. You lose a life if you are hit by a vehicle, fall in the water, or run out of time.

  • Move Up: Use the `Up Arrow` key.
  • Move Down: Use the `Down Arrow` key.
  • Move Left: Use the `Left Arrow` key.
  • Move Right: Use the `Right Arrow` key.

Core Features

  • Classic frog-hopping gameplay across roads and rivers.
  • Procedurally drawn graphics for all game objects, no external images needed!
  • A progressive level system with faster traffic and new hazards like snakes.
  • Complex water physics with moving logs and diving turtles.
  • Authentic CRT monitor simulation with scanlines and phosphorous glow for a true retro feel.
  • Scoring system that rewards forward movement, reaching home, and remaining time.

Setup & Customization

To hear the sound effects, you must place your own audio files in the same folder as the game's HTML file. The game looks for the following specific filenames:

  • hop.wav
  • squash.wav
  • splash.wav
  • home.wav
  • level-up.wav
  • music.wav

Developer Notes

The entire game is self-contained within the `index.html` file. All game logic, drawing functions, and styling are available to view and edit in one place. Game parameters such as frog speed, vehicle speeds, and starting lives can all be tweaked by changing the constant variables at the top of the script.
