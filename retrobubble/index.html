Description

Retro Bubble is a vibrant arcade platformer built in HTML5, inspired by the 1986 classic "Bubble Bobble". Take control of a cute, bubble-blowing dragon and trap your enemies. Jump on the bubbles to pop them, collect the fruit, and clear all 10 stages!

How to Play

  • Move Left/Right: Use the `Left & Right Arrow` keys.
  • Jump: Use the `Up Arrow` key.
  • Shoot Bubbles: Press the `Spacebar`.

Trap enemies in your bubbles, then jump on them to defeat them and get bonus items!

Core Features

  • Classic 2D platformer controls with "floaty" jumps and screen wrap-around.
  • Bubble-blowing mechanics to trap and defeat enemies.
  • 10 unique, single-screen levels with increasing difficulty.
  • Animated characters with custom-drawn pixel art sprites.
  • Themed levels with varied, patterned platforms for visual flair.
  • Authentic retro feel with CRT scanlines and a phosphor glow effect.
  • Dynamic sound effects generated with Tone.js.

Setup & Customization

To run the game, place all the asset files in the same folder as the main retro_bubble.html file. The game requires the following files:

  • player_sprites.png
  • monster_sprites.png
  • bubble_sprites.png
  • bonus_sprites.png

Developer Notes

The game's logic is contained within a single `<script>` tag in the `retro_bubble.html` file. Game parameters such as player speed, jump force, level layouts, and animation data can all be tweaked by changing the variables at the top of the script.
