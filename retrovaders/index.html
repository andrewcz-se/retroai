Description

Retrovaders is a retro arcade shooter built with HTML5 and Javascript. It is a homage to the arcade classic, Space Invaders, but with new and modern features. The player must defeat waves of aliens while avoiding enemy fire and using shields for protection.

How to Play

  • Move Left: Use the `Left Arrow` key.
  • Move Right: Use the `Right Arrow` key.
  • Shoot: Press the `Spacebar` to fire your laser.

Core Features

  • Classic, responsive player controls for movement and shooting.
  • Invaders that move in formation and increase speed as their numbers dwindle.
  • Destructible shields that provide temporary cover.
  • Randomly appearing mystery ship for bonus points.
  • Power-ups that grant temporary rapid-fire or invincibility shields.
  • A progressive level system with increasing difficulty.

Setup & Customization

To hear the sound effects, you must place your own audio files in the same folder as the game's HTML file. The game looks for the following specific filenames:

  • shoot.wav
  • background.wav
  • invader_killed.wav
  • player_death.wav
  • powerup.wav
  • mystery_ship.wav

Developer Notes

The game's logic is contained within the `<script>` tag in the `retroidvaders.html` file. Game parameters such as player speed, invader speed and laser speed can all be tweaked by changing the variables at the top of the script.
