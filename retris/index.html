Description

Retris is a fully-featured, retro-style version of the classic tile-matching puzzle game, built in HTML5. It captures the look and feel of 1980s arcade games with pixel-art graphics, authentic synthesized music and sound effects, and smooth, addictive gameplay.

How to Play

The goal is to manipulate falling blocks (Tetrominoes) to create complete horizontal lines. Completed lines disappear, earning you points. The game ends when the blocks stack up to the top of the screen.

Keyboard Controls:

  • Move Left/Right: `Left` / `Right` Arrow keys.
  • Rotate Piece: `Up` Arrow key.
  • Soft Drop (Faster Fall): `Down` Arrow key.
  • Hard Drop (Instant Fall): `Spacebar`.

Gamepad Controls:

  • Move: D-Pad Left / Right / Down.
  • Rotate Piece: `A` Button.
  • Hard Drop: `B` Button.
  • Start Game: `Start` Button.

Core Features

  • Authentic retro gameplay on a 10x20 grid.
  • Seven standard Tetromino shapes with vibrant, pixel-art styling.
  • Full keyboard and gamepad support for responsive controls.
  • Dynamic scoring system for single, double, triple, and "Tetris" line clears.
  • Progressive difficulty with increasing speed as you level up.
  • A rich, synthesized chiptune soundtrack and classic sound effects.
  • Visual flair including landing particles, 3D-beveled blocks, and custom UI frames.
  • Randomly changing backgrounds on each level-up for a fresh visual experience.

Setup & Customization

To experience all visual and audio features, you can place your own media files in the same folder as the game's `index.html` file. The game looks for the following specific filenames:

Sound Effects (.wav)

  • move.wav
  • rotate.wav
  • land.wav
  • line-clear.wav
  • tetris.wav
  • level-up.wav
  • game-over.wav

Developer Notes

The entire game is self-contained within the `index.html` file. All game logic, including parameters for colors, shapes, piece drop speed, and music notes, can be found and tweaked within the `<script>` tag at the bottom of the file.
